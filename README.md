# By Bash – Social Media Poster System

A collection of **social media posters designed for the “By Bash” identity**, created to be used across platforms such as **X, LinkedIn, Instagram, and newsletters**.

This project is part of the **By Bash digital identity**, exploring how AI tools and vibe coding can help automate design workflows.

---

## Concept

The poster system is inspired by the **Yin–Yang philosophy**, representing balance between:

* Technology and creativity
* Automation and design
* Simplicity and expression

The goal was to create a **consistent visual identity** that can easily be reused across different platforms.

---

## Posters

The system includes posters designed for:

* **X (Twitter)**
* **LinkedIn**
* **Instagram**
* **Newsletters**
* **General social announcements**

All posters follow the **By Bash visual identity** and maintain a minimal, modern aesthetic.

---

## Features

* Social media–ready poster templates
* Consistent **By Bash visual identity**
* **Yin–Yang inspired aesthetic**
* Designed for cross-platform use
* Automated design workflow using AI tools
* Reusable layouts and visual system

---

## Tools

Created using:

* **Figma AI**
* **Canva (initial exploration)**
